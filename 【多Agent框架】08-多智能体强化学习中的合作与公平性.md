## 概述

[多智能体系统](https://zhida.zhihu.com/search?content_id=251930048&content_type=Article&match_order=1&q=%E5%A4%9A%E6%99%BA%E8%83%BD%E4%BD%93%E7%B3%BB%E7%BB%9F&zhida_source=entity)被训练以最大化共享成本目标，这些目标通常反映了系统级的效率。然而，在移动性和交通系统的资源受限环境中，效率可能是以牺牲公平性为代价实现的——某些智能体可能会承担明显更大的成本或相比其他智能体获得更低的奖励反馈。任务分配的公平导致一些智能体获得不公平的优势，而其他智能体则承担过高的成本。因此，在这种设置中考虑**效率和公平性之间的权衡** 是非常重要的。

作者们考虑使用**多智能体强化学习（MARL）** 为一群分散的智能体解决公平的**多智能体导航问题，采用不同智能体旅行距离的变异系数的倒数作为公平性的度量** ，并研究智能体是否能够在不显著牺牲效率（即，增加总旅行距离）的情况下学会公平。

## 思路和方法

解决方案侧重于在合作化的分散式多智能体强化学习环境中，整个情节过程中公平性的演变。具体来说，设计一个奖励函数，它在训练中既鼓励任务的完成，也鼓励每个智能体在任务选择和执行中的总体公平性。所提出的奖励基于**[最小-最大公平距离目标分配](https://zhida.zhihu.com/search?content_id=251930048&content_type=Article&match_order=1&q=%E6%9C%80%E5%B0%8F-%E6%9C%80%E5%A4%A7%E5%85%AC%E5%B9%B3%E8%B7%9D%E7%A6%BB%E7%9B%AE%E6%A0%87%E5%88%86%E9%85%8D&zhida_source=entity)** 的组合，以及一个激励智能体在向目标移动时**保持公平性** 的项。本文的主要贡献包括：  
1、**引入了一个奖励函数** ，使底层的MARL算法能够学习智能体在导航完成覆盖任务时的公平行为。  
2、展示了一种**分散式的基于学习的目标分配方法** ，允许智能体在执行过程中适应性地选择它们的目标。  
3、**方法可以扩展** ，允许任意数量的智能体创建任何编队形状，而无需重新训练。

### 理论

作者研究的是一个**覆盖导航问题** ，其中一群智能体必须导航到不同的目标。有 个智能体在2D空间中使用[双积分动力学模型](https://zhida.zhihu.com/search?content_id=251930048&content_type=Article&match_order=1&q=%E5%8F%8C%E7%A7%AF%E5%88%86%E5%8A%A8%E5%8A%9B%E5%AD%A6%E6%A8%A1%E5%9E%8B&zhida_source=entity)进行导航。环境中有 个目标，每个目标由 \zeta ^{i} 表示。智能体在导航时，每个智能体都要到达一个独特的目标，同时避免障碍物和墙壁。任务是找到一个策略 \Pi =(\pi^{(1)},...,\pi^{(N)}) ，其中 \pi_{\theta}^{(i)} (a^{i} | o^{i} ,g^{i} ) 为智能体 根据其图网络和局部观测选择动作。

### 智能体训练框架

将训练步骤分组到多个情节中。在每个情节开始时，环境被初始化，智能体从**随机位置** 开始，目标在环境中**随机分布** 。策略为每个智能体 在第一步提供动作。智能体执行动作并观察状态的变化。每次，目标分配机制为智能体分配一个目标，并通过取负的欧几里得距离到这个分配的目标来计算一个初步的距离基础奖励 R_{dist}(s_{t}, a_{t}^{i}) 。此外，在奖励函数中加入一个[公平性指标](https://zhida.zhihu.com/search?content_id=251930048&content_type=Article&match_order=1&q=%E5%85%AC%E5%B9%B3%E6%80%A7%E6%8C%87%E6%A0%87&zhida_source=entity)，以创建一个“公平”的奖励。

智能体使用其感知半径来探测其附近的其他实体。利用这些实体创建一个图 g_{t}^{(i)} ，并将其输入到**[图神经网络](https://zhida.zhihu.com/search?content_id=251930048&content_type=Article&match_order=1&q=%E5%9B%BE%E7%A5%9E%E7%BB%8F%E7%BD%91%E7%BB%9C&zhida_source=entity) （GNN）**。GNN的输出 x_{agg}^{(i)} 和观测向量 o^{i} 被智能体用来选择其下一个动作。当智能体与其他智能体或障碍物相撞时，奖励中会加上一个**碰撞惩罚项** -C ;当一个智能体到达其分配的目标时，它会收到一次性的**目标奖励** R_{goal}(s_{t},a_{t}^{i}) 。这些智能体会被标记为“完成”并在当前情节结束前停留在目标处。此外，还限制了这些“完成”智能体可以采取的动作。这防止了智能体在其他仍在导航至各自目标的智能体的碰撞力影响下漂移。这种设置被称为“[死亡掩蔽](https://zhida.zhihu.com/search?content_id=251930048&content_type=Article&match_order=1&q=%E6%AD%BB%E4%BA%A1%E6%8E%A9%E8%94%BD&zhida_source=entity)”。当所有智能体到达它们的目标并“完成”时，结束情节并重新启动过程。

如上图示例，在导航场景中，随着情节的进展跟踪智能体的路径。

  * 帧A：情节开始时，智能体和目标被初始化到随机位置。为了便于表示，将它们沿着两条线排序。智能体1的观测向量 o^{(i)} 显示在绿色框中，其观测半径在蓝色阴影圆中突出显示。
  * 帧B和C：在每个时间戳，对于每个智能体，计算公平性指标 F_{t} 以及每个智能体的奖励。根据最优或公平的距离成本为智能体分配目标。
  * 帧D：一旦智能体到达分配的目标，它将获得目标奖励 R_{goal} ，并在该情节中被标记为“完成”。



### 重要公式及概念

**公平性指标公式：** F_{t}(s_{t},a_{t}^{(i)}) = \frac{\mu_{t}}{\delta_{t}+\epsilon}

公式解析：每个智能体 在每个时间步 旅行的距离由 d_{t}^{(i)} 表示，所有智能体的整体平均值为 \mu_{t} ，标准差为 \delta_{t} 。计算变异系数为 CV_{t} = \frac{\delta_{t}}{\mu_{t}} 。选择公平性指标 F = 1/  ，使其成为  的倒数，这样它就是一个无量纲量，数值越高表示公平性越大。为了避免分母为零的情况，在公平性指标 F 中加入一个小的正数 \epsilon 。

**公平性奖励函数公式：** R_{fair}(s_{t},a_{t}^{(i)})=\lambda tanh (F_{t}( s_{t},a_{t}^{(i)})-\tau_{0})

公式解析：通过一个公平性奖励函数将 F 传递给每个智能体策略 \pi^{(i)}  。公平性指标通过一个双曲正切函数进行转换，其零值通过 \tau_{0} 适当偏移，以使智能体拥有更高的公平性指标。这个值通过一个因子  进行缩放，以确定公平性奖励 R_{fair}(s_{t},a_{t}^{(i)}) 。总体奖励通过添加公平性奖励 （FR） 进行调整。

**每个时间戳的总奖励公式：** R_{total}(s_{t},a_{t}^{(i)})= R_{dist}(s_{t},a_{t}^{(i)})+R_{fair}(s_{t},a_{t}^{(i)})+\rho R_{total}(s_{t},a_{t}^{(i)})-\kappa c 。体智能体达到其分配的目标（由  表示）时，它会收到一次性的目标达成奖励 R_{total}(s_{t},a_{t}^{(i)}) 。如果智能体达到了分配的目标并且之前没有在分配的目标处，  为1；否则为0。 R_{fair}(s_{t},a_{t}^{(i)}) 为添加的公平性奖励， -c 为碰撞惩罚，来惩罚与环境中的其他智能体或障碍物相撞的智能体。 \kappa 是一个0/1变量，表示智能体是否与另一个智能体或障碍物相撞。

**本文用到的四种模型：**

  1. **随机目标分配** ，不包含公平性奖励**（RA）** 。随机化是常用的资源分配方法，用于提高公平性。这个模型作为基线模型。  
总奖励公式： R_{total}= R_{dist}^{RA}+\rho R_{goal}-\kappa c
  2. **最优距离成本目标分配** ，不包含公平性奖励**（OA）** ，使用线性和分配或二分图中的最小权重匹配算法实现高效分配。  
总奖励公式： R_{total}= R_{dist}^{OA}+\rho R_{goal}-\kappa c
  3. **最小-最大公平目标分配** ，不包含公平性奖励**（FA）** ，最小-最大公平分配通常用于网络带宽分配和交通调度问题等领域，防止单个智能体承担过大的负担。  
总奖励公式： R_{total}= R_{dist}^{FA}+\rho R_{goal}-K c
  4. **公平目标分配与公平性奖励（FA+FR），这是本论文提出的方法。**



总奖励公式： R_{total}= R_{dist}^{FA}+R_{fair}+\rho R_{goal}-K c

其中 \rho,K,-c 等含义参照上文。

### **实验设置**

在智能体可以导航至任何目标的覆盖任务中，作者评估了第二章提到过得四种模型，如下图所示：  


图1

图1所示四种导航模型的行为可视化，智能体从环境的上半部分开始，导航至位于环境左下部分的目标。在轨迹中，较深的阴影表示每个智能体旅行的较新状态，而较浅的圆圈表示较早的状态。  
三个智能体在环境的上部初始化，三个目标位于左下角。智能体根据它们训练的不同策略导航至目标，彩色点显示了它们的轨迹。最优距离分配是指最左边的智能体（智能体1）选择最近的目标，中间的智能体（智能体2）选择中间的目标，最右边的智能体（智能体3）选择下方的目标。然而，这种分配本质上是不公平的，因为它要求最右边的智能体覆盖比最左边智能体长得多的距离。一个公平的分配将允许位于最远的智能体选择离它最近的目标，即智能体3选择上方的目标，以此类推。在多智能体强化学习中的合作和公平性分配中，这样的分配将更加公平。

### **评估标准**

计算以下指标来确定方法的性能：

  1. **公平性 (F)** ：这是在情节结束时获得的总公平价值（公式1），用 F 表示（越高越好）。
  2. **情节时间比例 (T)** ：所有智能体达到其目标所需的情节时间的比例，用 T 表示（越低越好）。如果任何智能体没有达到其目标，则 T 设置为 1。
  3. **每情节的总旅行距离 (D)** ：每情节中智能体群体旅行的总距离，用 D 表示（越低越好）。
  4. **成功率 (S%)** ：能够到达唯一目标并变为‘完成’状态的智能体的百分比，用 S% 表示（越高越好）。



表1

表1展示了图一中所示的四种导航模型的**公平性指标 F、成功率 S%、情节时间比例 T 以及总旅行距离 D** 。除了随机目标分配（RA）模型，其他模型的智能体成功地达到了它们的目标，S%为100。可以看到，用随机目标分配（RA）训练的模型具有最高的公平性指标，但这是以牺牲效率为代价的且智能体也未能达到环境中的所有目标。然而，用公平目标分配和公平性奖励**（FA+FR）** 训练的模型在旅行总距离较低的情况下**实现了高公平性指标** 。用最优距离成本目标分配（OA）训练的模型实现了最有效率的行为，旅行距离最低，它的公平性指标也是最低的。这表明**（FA+FR）** 能够在整体效率仅略有下降的情况下**实现公平行为** 。

### **目标分配和公平性对于奖励的影响**

在这个实验中，对比**（FA+FR）模型** 与**其他三个模型** 在**3、5、7和10个智能体场景中** 100个测试轮次的中位数公平性指标F的改进，并且还计算了（FA+FR）模型与（OA）模型在总旅行距离D上的变化。

实验结论：在3、5、7和10个智能体场景中，**（FA+FR）模型** 的公平性指标平均比（OA）模型**提高了20.65±5%** ，而智能体旅行的总距离平均仅增加了6.86±6%，且**（FA+FR）** 在所有智能体数量上也**显著优于FEN** 。这表明，通过公平目标分配和公平奖励进行训练可以在不显著牺牲效率的情况下提高导航场景中的整体公平性。

### **拥堵对整体公平性和效率的影响**

该实验研究增加**智能体密度** 对智能体性能指标的影响，以及环境中的**障碍物和墙壁** 如何影响整体公平性。  


图2

图2的左半部分显示了一个增加了障碍物和墙壁的环境，其中有三名智能体。在右边，相同环境大小但有七名智能体。实体密度的增加导致了更多的智能体交互和与智能体数量较少的实验相比增加了碰撞。

实验结论：智能体在这种拥堵场景中，在最初的时间，智能体可能会优先考虑获得公平奖励而不是目标奖励，这可能导致智能体在目标周围移动，以减少旅行距离的偏差并优化它们的公平性指标。特别是在（FA+FR）模型中，智能体故意徘徊以提高公平性指标，以获得更高的公平奖励，以补偿缺乏容易、早期的目标奖励。这种行为突显了在保持效率的同时实现公平性的复杂性，尤其是在有许多实体的大型环境中。

## **总结与展望**

这篇论文使用多智能体强化学习解决了多智能体公平导航的问题。提出的模型（FA+FR）通过使用**最小-最大公平距离目标分配** 以及**一个奖励项** 来训练智能体，以激励它们在移动过程中保持公平。结果表明，智能体可以在不显著牺牲效率的情况下学习公平分配。此外，即使在比训练时更多的智能体数量上进行测试，模型也几乎实现了完美的目标覆盖，展示了方法的**可扩展性** 。

对于目标覆盖场景，与使用随机分配训练的基线模型相比，模型在效率上平均提高了14%（以智能体旅行的总距离来衡量），并且在公平性上平均提高了5%。此外，与最优分配且奖励中没有公平性的训练模型相比，（FA+FR）模型在公平性上平均提高了21%，但效率平均下降了7%。

方法实现了更高程度的去中心化，减少了对集中式Oracle的依赖。此外方法有一定的隐私保护，以及实施了“死亡掩蔽”来防止碰撞力影响目标达成行为。该方法能够推广到不同的编队形状，并在不重新训练模型的情况下实现完全覆盖。

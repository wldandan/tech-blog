2026年4月，Synthius团队发布了一篇重量级论文《[Synthius-Mem](https://zhida.zhihu.com/search?content_id=273243672&content_type=Article&match_order=1&q=Synthius-Mem&zhida_source=entity): Brain-Inspired Hallucination-Resistant Persona Memory Achieving 94.4% Memory Accuracy and 99.6% Adversarial Robustness on [LoCoMo](https://zhida.zhihu.com/search?content_id=273243672&content_type=Article&match_order=1&q=LoCoMo&zhida_source=entity)》，推出了Synthius-Mem——一个完全重构的AI人格记忆系统。在行业公认的LoCoMo基准测试上，这套系统实现了94.37%的记忆准确率和99.55%的对抗鲁棒性，不仅超过了目前所有同类产品，甚至比人类测试者的87.9 F1得分还要高。

## 一、AI记忆的痛点，我们都感受过

你有没有过这种经历：和AI助手聊了几十轮，交代了好几个重要需求，转头它就忘了其中一半？更糟的是有时候它会编出一套你根本没说过的信息，说的有鼻子有眼，你都要怀疑是不是自己记错了。

这不是模型不够聪明，是记忆系统的架构出了问题。当前所有主流方案都绕不开三个死穴：

  1. **成本爆炸** ：全上下文重放的token消耗随对话长度呈O(n²)增长，500轮对话每次请求就要吃掉2.6万token，普通人根本用不起
  2. **精度低下** ：滑动窗口、定期摘要、传统RAG这些方案，要么丢掉90%以上的历史信息，要么出现语义漂移，答非所问
  3. **毫无安全防护** ：几乎所有公开的记忆系统都没有对抗鲁棒性测试，随便挖个坑就能诱导它编造你从来没说过的隐私信息



在LoCoMo测试集上，即使是[GPT-4 Turbo](https://zhida.zhihu.com/search?content_id=273243672&content_type=Article&match_order=1&q=GPT-4+Turbo&zhida_source=entity)配4K窗口也只能拿到32.1 F1，之前最好的系统[MemMachine](https://zhida.zhihu.com/search?content_id=273243672&content_type=Article&match_order=1&q=MemMachine&zhida_source=entity)也只有91.69%准确率，而且完全不做对抗安全测试。

## 二、核心思路：向人类大脑抄作业

Synthius-Mem的突破说穿了很简单：别再把所有信息都塞到同一个向量库里了，学学人类大脑的记忆结构。

图1 Synthius-Mem系统架构分为上下两个部分：上半部分（提取流水线）：输入解析 → 分块 → 跨六个认知领域的并行领域提取 → 按类别合并 → 冷存储；下半部分（聊天时检索流程）：用户消息 → 规划大语言模型 → CategoryRAG → 回答大语言模型 → 回复。其中 CategoryRAG 查询冷存储的平均延迟为 21.79 毫秒

[神经科学](https://zhida.zhihu.com/search?content_id=273243672&content_type=Article&match_order=1&q=%E7%A5%9E%E7%BB%8F%E7%A7%91%E5%AD%A6&zhida_source=entity)研究早就证明，人类记忆不是一个统一的存储池，而是由多个功能特化的子系统组成的：记生平事实的语义记忆、记人生经历的情景记忆、记好恶的评价记忆、记人际关系的社会认知系统，都是完全独立的神经通路，存储和提取机制完全不一样。

图2 六领域记忆模型，每个领域将一个认知科学概念映射为具有独立模式、提取模块和检索工具的计算表示。

Synthius-Mem第一次把这个原理落地到了工程实现里，把人格记忆划分成了**六大[认知域](https://zhida.zhihu.com/search?content_id=273243672&content_type=Article&match_order=1&q=%E8%AE%A4%E7%9F%A5%E5%9F%9F&zhida_source=entity)**：

认知域| 对应大脑记忆类型| 存储内容  
---|---|---  
生平信息| 语义自我知识| 个人基本信息、教育、家庭、健康等19个类别原子事实  
经历记忆| 情景记忆| 结构化的人生事件，包含时间、情绪强度、社交上下文  
偏好记忆| 评价记忆| 好恶判断，包含极性、强度、时间状态和原始表述  
社交圈| 社会认知| 人际关系模型，包含亲密度、信任度、关系事件  
工作记忆| 职业记忆| 职业经历、技能、工具使用、项目、成果  
心理测量| 自我模型/元认知| 9种经过验证的心理学框架的标准化评分和证据  
  
这不是简单的标签分类。每个认知域都有独立的提取模块、合并逻辑和检索工具，用户问人际关系问题就只搜社交圈域，问职业问题就只搜工作域，不用在整个无差别语料库里大海捞针。

整个架构遵循三个核心设计原则：

  1. **领域结构化存储** ：不同类型信息匹配专门的存储模式和Schema，不搞一刀切
  2. **主动巩固** ：类似大脑的记忆巩固过程，系统自动去重、解决冲突、层级重构
  3. **Token预算约束** ：严格限制每一步的token消耗，再长的对话也能在固定计算成本内处理



## 三、运行逻辑：从对话到记忆的完整流水线

整套系统分为**离线处理** 和**在线查询** 两个阶段。

**离线阶段**

负责把无结构的对话转化为结构化记忆：

  1. 先做输入解析，支持WhatsApp、Telegram、邮件、PDF等各种格式；
  2. 然后按token限制分块重叠处理，避免信息丢失；
  3. 接着并行对六个认知域做结构化提取，输出对应格式的JSON数据；
  4. 再做分类巩固，按类别去重、解决冲突，确保信息的语义完整性；
  5. 最后持久化到冷存储，自动生成生平叙事和人生事件时间线。



**在线查询阶段**

该阶段的速度快得惊人：

  1. 用户提问后，先由规划LLM判断需要查询哪些认知域；
  2. 然后调用对应域的检索工具做字段级匹配检索（这套机制叫做**CategoryRAG** ）；
  3. 最后把检索到的结构化事实送入回答LLM生成回复。



整个检索过程的平均延迟只有**21.79毫秒** ，比传统RAG快了几个数量级，精度反而更高。

记忆更新也做了特殊设计：通过可逆的差异引擎支持添加、编辑、删除操作，完整的回滚能力确保记忆更新不会出问题。

## 四、实测成绩：比人类记得更准

在LoCoMo基准测试上，Synthius-Mem的成绩甩开了所有竞品一大截：

  * 整体准确率**94.37%** ：超过第二名MemMachine的91.69%，比人类测试者的87.9 F1还要高；
  * 核心记忆事实准确率**98.64%** ：姓名、住址、职业这些定义身份的关键事实几乎不会记错；
  * 对抗鲁棒性**99.55%** ：这是业内第一个公开对抗鲁棒性指标的记忆系统，几乎不会编造用户从未提及的信息；
  * Token效率比全上下文重放低**5倍** ：精度更高的同时，成本只有传统方案的1/5。



有意思的是，这套系统根本不是为了跑分开发的。六大认知域、19个生平分类、9个心理测量框架，所有的Schema和逻辑都是为生产环境设计的，已经在500MB真实用户对话数据上跑了很久。LoCoMo测试只是事后的性能验证，不存在为了跑分做过拟合的问题。

## 五、落地价值：真正的个性化AI终于成为可能

Synthius-Mem的突破让很多之前想都不敢想的应用变成了现实：

  * 个人AI助理可以真正记住你的所有偏好、经历、人际关系，和你聊几年都不会“失忆”，不用每次用都重新交代一遍背景
  * 客户服务机器人能记住每个客户的历史需求、偏好、投诉记录，不用每次打电话都让你复述一遍问题
  * 心理健康陪伴机器人可以长期跟踪用户的情绪状态、经历变化，提供更有针对性的支持
  * 企业知识助理能记住每个员工的技能、项目经历、知识结构，提供完全个性化的工作支持



尤其是**99.55%的对抗鲁棒性** ，这个指标的意义比准确率还要大。它意味着系统几乎不会被诱导编造用户的隐私信息，这对涉及个人敏感数据的应用来说是生命线。

## 六、一点思考

当然Synthius-Mem也不是万能的，它现在只专注于人格记忆领域，通用知识记忆、程序性技能记忆这些场景还不适用。但它开辟的“认知科学启发的结构化记忆”方向，给整个行业指了一条新路。

现在大家都在卷模型参数、卷上下文窗口，但记忆能力其实是AI助手真正落地的核心瓶颈。什么时候AI能真正记住你是谁，记住你们之间发生过的所有事情，它才能真正成为你可信的长期伙伴。Synthius-Mem让我们看到了这个未来的影子。
